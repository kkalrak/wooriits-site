# 우리ITS(주) 회사 소개 웹사이트

우리아이티에스주식회사의 공식 웹사이트입니다. 교통정보시스템 및 교통신호제어기 전문 기업으로서 회사 소개, 제품 정보, 시공 실적 등을 안내합니다.

## 회사 소개

- **회사명**: 우리아이티에스주식회사
- **대표이사**: 진미라
- **설립일**: 2011년 02월 22일
- **소재지**: 경상남도 창원시 성산구 연덕로 15번길 16 (중앙동, 아이케이빌딩 401호)
- **사업분야**: 정보통신공사, 교통신호제어기, LED신호등, CCTV, 교통정보시스템 서비스업

## 웹사이트 구조

이 웹사이트는 다음과 같은 섹션으로 구성되어 있습니다:

- **회사소개**: 기본 회사 정보 및 사업 분야
- **연혁**: 2011년 설립부터 현재까지의 주요 이력
- **조직도**: 경영지원, 기술영업부, 기술지원 등 조직 구조
- **사업분야**: ITS사업, 교통신호기, 교통정보시스템
- **제품소개**: WRLC-S1000 교통신호제어기 상세 사양
- **인증서**: 사업자등록증, 정보통신공사업 등록증 등
- **시공실적**: 주요 프로젝트 실적
- **연락처**: 연락처 및 위치 정보

## 기술 스택

- **HTML5**: 시맨틱 마크업
- **CSS3**:
  - CSS Variables를 활용한 테마 관리
  - Flexbox/Grid 레이아웃
  - 반응형 디자인 (모바일, 태블릿, 데스크톱)
  - CSS 애니메이션
- **Vanilla JavaScript**: 모바일 네비게이션 토글 기능

## 주요 기능

- 반응형 디자인으로 모든 디바이스 지원
- 부드러운 스크롤 애니메이션
- 접근성을 고려한 네비게이션
- 프린트 최적화 스타일

## 설치 및 실행

이 웹사이트는 정적 HTML 페이지로 구성되어 있어 별도의 빌드 과정이 필요하지 않습니다.

### 로컬에서 실행

1. 저장소 클론 또는 파일 다운로드
2. `index.html` 파일을 웹 브라우저로 열기

### 간단한 로컬 서버 실행 (선택사항)

```bash
# Python 3 사용 시
python -m http.server 8000

# Node.js http-server 사용 시
npx http-server
```

브라우저에서 `http://localhost:8000` 접속
